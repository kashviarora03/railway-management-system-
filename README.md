# 🚆 Railway Management System

The Railway Management System is a database-driven desktop application developed using **Python**, **Tkinter**, and **MySQL** to streamline and automate railway management operations. The application provides a simple, interactive graphical user interface that allows users to efficiently manage various aspects of a railway system, including passenger information, train details, ticket reservations, payment records, technical supervisors, and train inspections.

The system enables users to add, update, and view passenger records, maintain train information, issue tickets for passengers, record payments, assign technical supervisors to trains, and keep track of passenger-train inspection records. All data is securely stored and managed in a MySQL database, ensuring consistency, reliability, and efficient retrieval of information. The application uses Python's `mysql-connector` library to establish seamless communication between the graphical interface and the database, allowing real-time execution of database operations.

The graphical user interface has been developed using the Tkinter library, making the application user-friendly and easy to navigate. Each module is designed to perform specific CRUD (Create, Read, Update, Delete) operations, demonstrating effective integration of front-end interface design with back-end database management. The project follows a modular approach, making the code organized, maintainable, and easy to extend with additional features.

This project demonstrates practical implementation of core Database Management System (DBMS) concepts such as relational database design, primary and foreign keys, normalization, SQL queries, and table relationships. It also showcases Python programming concepts including functions, exception handling, database connectivity, GUI development, and modular programming. By combining these technologies, the application provides an efficient solution for managing railway operations while serving as a comprehensive academic project that reflects real-world software development practices.

### Technologies Used

* Passenger Management

This project was developed as part of a Database Management System (DBMS) course to demonstrate the integration of Python programming with MySQL for building a fully functional desktop application capable of managing railway operations efficiently.
